# LOA ON SUMMER — English Countdown

An unofficial English-language countdown page for the **2026 LOA ON SUMMER**
Lost Ark showcase, mirroring the official announcement page (ships + ocean
background) with English text and a live countdown.

- **Live:** https://thejunglewalrus.github.io/loaon-countdown/
- **Event:** June 20, 2026 · 16:00 KST · 3:00 AM ET (`2026-06-20T07:00:00Z`)
- **Official page:** https://lostark.game.onstove.com/Event/Promotion/LOAON/260603

## How it works

Static single page — no build step. `index.html` overlays an English
LIVE/date badge, a JavaScript countdown, and broadcast-alert buttons on top of
the official looping background video (`assets/loaon.mp4`). The countdown target
is hard-coded in the inline script.
